Programas aprendidos



C++

Es uno de los lenguajes de programación más populares del mundo. Se puede encontrar en los sistemas operativos, interfaces gráficas de usuario y sistemas embebidos actuales. Es un lenguaje de programación orientado a objetos que proporciona una estructura clara a los programas y permite reutilizar el código, lo que reduce los costos de desarrollo.

JAVA

Java es el lenguaje de Programacion mas versatil. Es Orientada a objetos, este lenguaje es elegido para muchas cosas como: Desarrolo de aplicaciones Android, Desarrollo web Back. end

HTML5

HTML5 es un lenguaje markup que se utiliza para estructurar y mostrar contenidos en web. Proporciona soporte para aplicaciones multimedia avanzadas, incluyendo audio y vídeo. También permite modificar el layout de una página y modificar así su aspecto.
